# StencilForge v1

Flat GitHub-ready PWA for making layered SVG airbrush stencils.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `README.md`

Upload these files directly to the root of a GitHub repo. Do not put them inside an extra folder.

## What works in v1
- Upload a photo.
- Describe an idea and generate a local starter sketch.
- Change the local idea style.
- Edit before making layers with Keep and Remove brushes.
- Choose 2 to 12 layers.
- Pick colors for each layer.
- Auto-connect floating stencil islands with adjustable bridge thickness.
- Add registration marks for lining up layers.
- Export each layer as SVG.
- Export combined SVG.
- Save/load project in browser localStorage.
- PWA install support.

## Important note
The text description feature is local only in this version. Real AI image generation requires a backend/API key later so the key is not exposed in the browser.
