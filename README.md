## What changed in v2
- Better contrast and tone preprocessing
- Edge detail boost
- Keep/remove brush before stencil generation
- Auto background fade
- Improved multi-layer posterized stencil generation
- Auto-connect floating islands with bridge thickness control
- Visible registration marks on every layer
- Individual layer colors
- Separate SVG downloads
- Combined SVG download
- PWA install support
